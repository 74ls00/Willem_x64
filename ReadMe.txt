Скрипт полуавтоматической установки драйвера TVicPort LPT для Windows x64
Предназначен для использования программы как портативной, без траты времени на поиск и установку драйвера.

Определив разрядность системы если она x64, этот способ только делает проверку наличия exe файла в папке драйвера по пути указанному в реестре.
Если файл не обнаружен, запускается установщик драйвера,  также в консоле отображается результат поиска других файлов.
Завершение установки драйвера требует перезагрузки.

Описание файлов:
devmgmt.bat запуск диспетчера задач в возможностью показа скрытых уствойст.
  Меню "Вид - Показать скрытые устройства"  "Драйверы несамонастраиваемых устройств"+"TVicPort64"
io.dll библиотека драйвера, адрес порта указывается в io.ini, для LPT1 это 0x378
kill.bat аварийное завершение программы EpromM51_98D12C3.exe которая была запущена без драйвера порта.
TVicPortInstall41.exe установщик драйвера
